# FeedbackPulse

FeedbackPulse is a modern, full-stack application designed to help you collect, manage, and analyze user feedback for your projects with ease. Built with Next.js, it features an intuitive dashboard, embeddable feedback widgets, and AI-powered sentiment analysis.

## Key Features

- **Project Management**: Create and manage multiple projects to keep feedback organized.
- **Embeddable Widget**: Easily generate and embed a feedback widget onto any website to start collecting user input.
- **Categorized Feedback**: Users can categorize their feedback as a bug report, a feature request, or other comments.
- **AI-Powered Sentiment Analysis**: Automatically analyze the sentiment (positive, negative, neutral) of feedback comments to gain insights quickly.
- **Custom Labeling**: Add custom tags and labels to feedback submissions for better organization and filtering.
- **Clean Dashboard**: A central dashboard provides an at-a-glance overview of all your projects and their feedback volume.
- **Responsive Design**: A clean, modern, and fully responsive user interface built with Tailwind CSS and ShadCN UI components.

## Getting Started

To get this project up and running on your local machine, follow these steps.

### Prerequisites

You will need to have Node.js and npm installed.

### Installation & Running the App

1.  **Install dependencies:**
    ```bash
    npm install
    ```

2.  **Run the development server:**
    ```bash
    npm run dev
    ```

3.  Open [http://localhost:9002](http://localhost:9002) in your browser to see the application. The app will automatically redirect you to the login page.
